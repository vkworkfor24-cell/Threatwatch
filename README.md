🛡️ ThreatWatch

ThreatWatch is an intelligent web-based security scanner that helps identify common vulnerabilities and security misconfigurations in websites. It provides automated security assessments and detailed scan reports to improve web application security.

🚀 Features

🔗 Scan target websites using a URL

🛡️ Detect missing HTTP security headers

↩️ Identify Open Redirect vulnerabilities

💉 Perform basic SQL Injection (SQLi) checks

🌐 Detect CORS misconfigurations

📡 Conduct basic SSRF (Server-Side Request Forgery) tests

⚠️ Perform basic reflected XSS detection

💾 Store scan results in a database for future reference

📊 Generate detailed vulnerability reports

⚡ Fast and easy-to-use interface

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Security Testing: Axios, Custom Vulnerability Detection Modules

Environment Management: .env

Version Control: Git, GitHub

📂 Project Overview

ThreatWatch automates the process of identifying common web vulnerabilities and security weaknesses. By entering a target URL, users can perform security scans and obtain actionable insights to enhance website protection.

🎯 Supported Vulnerability Checks
HTTP Security Headers Analysis
Open Redirect Detection
SQL Injection (SQLi) Detection
Cross-Origin Resource Sharing (CORS) Misconfiguration Detection
Server-Side Request Forgery (SSRF) Testing
Reflected Cross-Site Scripting (XSS) Detection
Scan History Management and Report Storage
▶️ Usage
Open the application in your browser.
Enter the target website URL 🌐
Start the security scan 🔍
ThreatWatch analyzes the target for vulnerabilities 🛡️
View the scan results and vulnerability reports instantly 📊
